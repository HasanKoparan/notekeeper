# NoteKeeper

A JavaScript note-taking app built with HTML, CSS, and JavaScript.

## Features
- Add new notes with title and description
- Modal popup interface
- Modern gradient UI design

## Technologies Used
- HTML5
- CSS3
- JavaScript
